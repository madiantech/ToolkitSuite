# ToolkitSuite

## 概要

ToolkitSuite是TKCore官方提供的、旨在提高开发效率的一套辅助软件。常用的功能有应用生成工具、模块分发工具、代码生成工具。 [工具介绍资料](http://www.tkcore.net/toolkitsuite/)


## 应用生成工具

快速创建TKCore Web应用的一款工具，不用一一引用DLL、不用一一创建框架固有的文件夹，一键生成！


## 模块分发工具

模块分发可以添加通用的模块到应用。


## 代码生成工具

一键生成DataXml、ModuleXml！
